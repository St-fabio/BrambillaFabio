Utilisation d'une base de donnée

Description :

Ce projet est un projet de l'IUT, dont l'objectif été d'utilisé ube base de donnée open source pour répondre à un sujet donné.
Notre sujet concerné le lien entre le nutriscore et les valeurs nutritionnelles des produits céréaliers au USA.
Cela c'est fait en deux partie, dabord une partie de trie de la base où l'on a prix seulement les info que l'on recherché. Puis une partie en R pour les stats.
